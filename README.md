# baekJoonProblemPractice

백준 문제 풀이 저장소입니다.

## 개요
- 언어: C++ (일부 C)
- 개발 환경: Xcode 프로젝트(`.xcodeproj`) 중심
- 분류: 문제 유형별 디렉터리 구성

## 디렉터리 구조
- `IO_and_BODMAS` (8문제): 입출력과 사칙연산
- `if_Statement` (5문제): 조건문
- `for_Statement` (11문제): for문
- `while_Statement` (3문제): while문
- `1DArray` (7문제): 1차원 배열
- `Method` (3문제): 함수
- `String` (10문제): 문자열
- `basicMath` (2문제): 기본 수학
- `Image`: 학습 캡처 이미지
- `output`: 실행 결과/출력 관련 폴더

현재 `main.cpp` 기준 풀이 수: **49개**

## 파일 네이밍 규칙
대부분 아래 형식을 따릅니다.
- `{문제번호}_{YYMMDD}_{문제이름}`

예시:
- `1000_220114`
- `10818_220118_maxAndMin`

## 실행 방법
1. 원하는 문제 폴더의 `.xcodeproj`를 엽니다.
2. `main.cpp`를 빌드/실행합니다.

또는 C++ 컴파일러가 있다면:

```bash
g++ main.cpp -o main
./main
```
