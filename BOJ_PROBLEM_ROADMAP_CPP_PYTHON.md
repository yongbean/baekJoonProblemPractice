## 언어 운영 규칙

- 홀수 번호 문제: C++ 우선 -> Python 재풀이
- 짝수 번호 문제: Python 우선 -> C++ 재풀이
- 각 문제마다 `복잡도`, `핵심 아이디어`, `실수 1개`를 메모로 남김
- 새 문제를 추가할 때는 같은 형식으로 행을 추가하고 `문제 시도`를 `[ ] 예정`으로 시작
