# phase0 review

## BOJ 2588 - 곱셈
- 특징: 자리수 분해 계산 연습
- 고려사항: 일의/십의/백의 자리 추출 정확성
- 고려사항: 중간 결과 출력 형식 유지

## BOJ 1000 - A+B
- 특징: 기본 입출력 구조 학습 문제
- 고려사항: 정수 입력 2개 파싱 정확성
- 고려사항: 불필요한 출력 문자열 제거

## BOJ 1001 - A-B
- 특징: 기본 사칙연산 출력 문제
- 고려사항: 음수 결과 가능성 고려
- 고려사항: 개행 처리 일관성 유지

## BOJ 1008 - A/B
- 특징: 실수 나눗셈 처리 문제
- 고려사항: 정수 나눗셈으로 계산하지 않기
- 고려사항: 출력 정밀도 요구 확인

## BOJ 10172 - 개 출력
- 특징: 이스케이프 문자 연습
- 고려사항: 백슬래시/따옴표 이스케이프 처리
- 고려사항: 줄바꿈 개수 정확히 맞추기

## BOJ 10430 - 나머지
- 특징: 모듈러 연산 성질 검증
- 고려사항: 각 식을 문제 순서대로 출력
- 고려사항: 정수형 처리 유지

## BOJ 10869 - 사칙연산
- 특징: 여러 연산 결과를 순서대로 출력
- 고려사항: 연산 순서와 출력 순서 구분
- 고려사항: 나눗셈/나머지 연산 규칙 확인

## BOJ 10998 - A*B
- 특징: 곱셈 연산 정확도 확인
- 고려사항: 입력 형식 그대로 읽기
- 고려사항: 오버플로우 범위 점검(확장 문제 대비)

## BOJ 18108 - 1998년생인 내가 태국에서는 2541년생?!
- 특징: 단순 수식 변환 문제
- 고려사항: 상수 오프셋(543) 정확히 적용
- 고려사항: 입력 1개, 출력 1개 패턴 유지

## BOJ 10171 / 10718 (misc ASCII 출력 문제)
- 특징: ASCII 아트/문자열 고정 출력으로 출력 포맷 정확도가 핵심
- 고려사항: 이스케이프 문자(`\\`, `\"`)와 개행 수를 정확히 맞출 것
- 고려사항: 불필요한 공백/문자 추가 출력 금지
